# VLAN Network Lab - Cisco Packet Tracer

Proyecto práctico de segmentación de red mediante VLANs realizado en Cisco Packet Tracer.

## Objetivo

Simular una red empresarial separando departamentos en diferentes VLANs para mejorar la organización y segmentación de tráfico.

## Tecnologías utilizadas

* Cisco Packet Tracer
* VLANs
* Switching
* Networking

## Topología

* 1 Switch Cisco 2960
* 6 PCs

## VLANs configuradas

| VLAN    | Departamento |
| ------- | ------------ |
| VLAN 10 | HR           |
| VLAN 20 | IT           |
| VLAN 30 | SALES        |

## Funcionalidades implementadas

* Creación de VLANs
* Asignación de puertos
* Segmentación de red
* Configuración de IPs
* Verificación de conectividad
* Troubleshooting básico

## Resultados

* Comunicación correcta dentro de cada VLAN
* Bloqueo de comunicación entre VLANs diferentes

## Conocimientos aplicados

* VLAN configuration
* Switch administration
* Network segmentation
* Connectivity testing
* Troubleshooting
